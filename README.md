# Metre AI — Mappa Sala

Dashboard interattiva per la gestione della sala ristorante.

## Funzionalità
- Mappa visiva della sala con tavoli colorati
- Verde = libero · Rosso = occupato
- Switch tra 1° turno (19:30) e 2° turno (21:30)
- Prenotazione rapida con un tocco sul tavolo
- Gestione stati: Prenotato, Arrivato, Completato, No-show, Cancellato
- Editor layout per posizionare i tavoli
